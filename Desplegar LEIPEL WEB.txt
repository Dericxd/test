*** Ambiente y Versiones proyecto Leipel web ***

|------------------------------------------------------------------------------------------------------------------------------------
|1- Composer version 1.6.5 o mayor
|------------------------------------------------------------------------------------------------------------------------------------
|2- PHP 7.1.16 o mayor
|------------------------------------------------------------------------------------------------------------------------------------

*** Despliegue  proyecto Leipel web ***

|------------------------------------------------------------------------------------------------------------------------------------
|1- Clonar proyecto. -> | git clone ssh o http | 
|------------------------------------------------------------------------------------------------------------------------------------
|2- Hacer el sigiente comando ->| composer install| por el terminal, dentro del proyecto. 
|------------------------------------------------------------------------------------------------------------------------------------
|3- Cambiar el archivo que esta dentro del proyecto que dice env por | .env | donde se configura la Base de datos y otros.
|------------------------------------------------------------------------------------------------------------------------------------
|4- Crear y cambiar de rama-> por defecto el crea la rama  (master) por lo tanto debes cambiar a la rama (tu nombre). 
|------------------------------------------------------------------------------------------------------------------------------------
